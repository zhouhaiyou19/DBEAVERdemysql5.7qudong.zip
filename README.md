## 如何使用

1. **下载驱动**：首先，点击下载“DBEAVER的mysql5.7驱动.zip”文件。
2. **解压缩**：将下载的zip文件解压至本地目录。
3. **配置DBEAVER**：
   - 打开DBEAVER，创建或选择一个MySQL 5.7的数据源。
      - 在数据源编辑器中，找到“Driver files”（驱动文件）部分。
         - 点击添加，从您解压的文件夹中选择对应的.jar文件。
